Status of the Node.js Buffer API

Reference Document: http://nodejs.org/api/buffer.html

Nodyn’s buffers differ slightly from those in node.js in that they are not a fixed size. They will grow to accommodate any overflow. This has some implications on a few of the methods. For example, Buffer.concat ignores the totalLength parameter, and buff.copy doesn’t fail when you exceed the length of the buffer.
