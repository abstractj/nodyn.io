Status of the Node.js HTTP API

Reference Document: http://nodejs.org/api/http.html

Nodyn’s implementation of the Node.js HTTP API is mostly complete. There are, however a few oustanding issues. 1) There is not yet any support for http.Agent and http.globalAgent. 2) Sockets for IncomingMessage.socket and ClientRequest socket events are not available yet, as Vert.x does not expose the underlying socket object through its HTTP API. 3) Connection upgrades with the ClientRequest.upgrade and Server.upgrade events are not yet supported.
