NodeJ

NodeJ is a node.js compatible framework, running on the JVM powered by the DynJS Javascript runtime running under vert.x - the polyglot application platform and event bus.

Imagine running your Node.js app on the JVM with clean, clear access to Java directly in your Javascript. All that with vert.x's powerful clustering technology built right in.

Open Source

NodeJ is an open source project, distributed with the Apache license and is copyrighted by the original authors. You can find the NodeJ source code on GitHub.

Related Projects

NodeJ is dependent on a couple of other awesome open source projects.

DynJS
The Javascript runtime that makes all of this work.
Vert.x
The polyglot, massively scalable application platform.
